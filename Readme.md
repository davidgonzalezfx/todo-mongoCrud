# MongoDb Crud

Simple ToDo web app that use mongoDb CRUD

### Tools:
* NodeJS
* MongoDb
* Express
* EJS

### How to use:

To use the project you must have NodeJs and MongoDb installed.

> Clone or download repo
> Open teminal
> Go to project folder
> Execute: npm install
> Execute: npm start
> Enjoy
